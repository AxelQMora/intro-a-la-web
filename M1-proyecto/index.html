DEV.F

Todos los términos que necesitas saber sobre del Módulo 1

Inteligencia Artificial

Inteligencia artificial

Rama de la informática que desarrolla sistemas capaces de realizar tareas que imitan la inteligencia humana, como aprender, razonar y tomar decisiones.

Prompt

Instrucción o mensaje que guía a un modelo de inteligencia artificial para generar una respuesta o realizar una tarea específica.

Large Language Model (LLM)

Modelo de IA entrenado con grandes cantidades de texto para procesar, comprender y generar lenguaje de forma coherente y natural.

Herramientas locales

Integrated Development Environment(IDE)

Software que agrupa herramientas para escribir, depurar y ejecutar código, facilitando el desarrollo de programas.

Visual Studio Code

Editor de código gratuito y ligero, desarrollado por Microsoft, con soporte para múltiples lenguajes y extensiones.

Chromium Project

Proyecto de código abierto que sirve como base para navegadores web como Google Chrome, proporcionando el motor y herramientas esenciales.

Inspector Web

Herramienta en navegadores que permite analizar y modificar el código fuente de una página web en tiempo real.

Terminal, Git y GitHub

Terminal

Interfaz de línea de comandos que permite a los usuarios interactuar con el sistema operativo mediante la ejecución de comandos escritos, en lugar de utilizar una interfaz gráfica.

Git Bash

Aplicación que proporciona una interfaz de línea de comandos para usar Git en sistemas Windows. Ofrece comandos similares a los de Unix, permitiendo gestionar proyectos y repositorios de código de manera eficiente.

Git

Sistema de control de versiones distribuido que permite gestionar y rastrear cambios en archivos de un proyecto, facilitando la colaboración entre múltiples desarrolladores.

GitHub

Plataforma en línea que permite alojar y gestionar repositorios de código utilizando Git. Facilita la colaboración, el control de versiones y la revisión de código entre desarrolladores.

Conventional Commits

Convención para escribir mensajes de confirmación (commits) en Git de manera estándar, que facilita la comprensión del historial de cambios.

HTML y CSS

HTML(HyperText Markup Language)

Lenguaje de marcado utilizado para estructurar y presentar contenido en la web. Define la estructura de una página web mediante etiquetas, como h1, p, y a, que indican encabezados, párrafos y enlaces, respectivamente.

CSS (Cascading Style Sheets)

Lenguaje utilizado para describir la presentación de un documento HTML. Permite controlar el diseño, los colores, las fuentes y otros aspectos visuales de una página web, mejorando la apariencia y la disposición de los elementos.

Landing page

Página web diseñada específicamente para convertir visitantes en clientes o usuarios, generalmente enfocada en una sola acción, como completar un formulario, suscribirse a un servicio o realizar una compra.

Header

Sección superior de una página web que generalmente contiene el logotipo, el menú de navegación y otros elementos clave como el título o una barra de búsqueda. Sirve como punto de entrada y facilita la navegación del usuario.

Footer

Sección situada al final de una página web que suele contener información adicional, como enlaces a políticas de privacidad, términos de uso, contacto y redes sociales, así como créditos o derechos de autor.

Article

Elemento de HTML utilizado para representar contenido independiente y autónomo, como publicaciones de blog, noticias o ensayos. Está diseñado para ser reutilizado o sindicado, y puede contener encabezados, párrafos, imágenes y otros elementos multimedia.

Aside

Elemento de HTML utilizado para representar contenido relacionado pero independiente del contenido principal de la página. Generalmente se usa para barras laterales, notas al margen o enlaces relacionados, y puede contener información adicional o complementaria.

Selectores

En CSS, los selectores son patrones utilizados para seleccionar y aplicar estilos a los elementos HTML específicos. Pueden ser simples (como seleccionar por etiqueta, clase o id) o complejos (utilizando combinaciones de estos o pseudoclases/pseudo-elementos).

Declaración

En CSS, una declaración es un conjunto de propiedades y valores que se aplican a un selector. Se escribe dentro de llaves {}, y tiene la forma propiedad: valor;. Por ejemplo, color: red; es una declaración que cambia el color del texto a rojo.

Box Model

Concepto en CSS que describe la estructura de un elemento HTML. Cada elemento se representa como un "caja" compuesta por contenido, relleno (padding), borde (border) y margen (margin), que afectan el tamaño y la disposición del elemento en la página.

Float

Propiedad de CSS utilizada para posicionar un elemento a la izquierda o derecha dentro de su contenedor, permitiendo que el contenido fluya alrededor de él. Se utiliza comúnmente en diseños de columnas o para ajustar la alineación de imágenes y texto.

Position

Propiedad de CSS que define cómo se posiciona un elemento en la página.
